Lembretes importantes

  • Toque no nome do exercício para abrir os 2 vídeos: frontal e lateral.
  • Use o botão 🦵 Músculo para abrir a imagem da região trabalhada.
  • Use 📝 Peso para salvar a carga do dia.
  • O histórico fica salvo no próprio aparelho via armazenamento local.
  • Com o PWA instalado no iPhone, a experiência fica mais parecida com app nativo.
